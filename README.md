
# uBudget

uBudget is a budgeting and financial tracking app designed to help users, especially students, manage their finances. With uBudget, users can track their income, monitor expenses, set and track savings goals, and keep an eye on debts. Built with React Native, uBudget provides an accessible and easy-to-use tool for maintaining financial health.

## Features

- **Income and Expense Tracking**: Easily add and categorize income and expenses to stay on top of your finances.
- **Savings Goals**: Set savings targets and track progress toward each goal.
- **Debt Management**: Record any debts and monitor repayments.
- **Spending Insights**: Visual reports to help users understand where their money is going.
- **Budget Reminders**: Set alerts to stay within budget and avoid overspending.

## Technology

- **React Native**: For cross-platform compatibility, allowing uBudget to run on both iOS and Android.
- **Redux**: To manage app state for smooth performance.
- **Firebase (optional)**: Used for data storage and user authentication if implemented in later versions.
- **Reanimated**: For smooth animations and transitions in the UI.

## Getting Started

### Prerequisites

- Node.js
- Yarn or npm
- Expo CLI (for development and testing)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ubudget.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ubudget
   ```

3. Install dependencies:
   ```bash
   yarn install
   ```

4. Start the project using Expo:
   ```bash
   npx expo start
   ```

### Running on Emulator or Device

- **iOS**: Run on the iOS simulator through Xcode or Expo Go on an iOS device.
- **Android**: Run on an Android emulator through Android Studio or Expo Go on an Android device.

## Usage

1. **Create a Budget**: Set up your monthly or weekly budget by adding income and allocating expenses.
2. **Track Your Spending**: Add transactions as you spend to get a real-time view of your financial status.
3. **Manage Savings and Debts**: Set up savings goals and add any debts to keep track of repayments.
4. **View Reports**: Check spending insights to see where you’re overspending or saving.
